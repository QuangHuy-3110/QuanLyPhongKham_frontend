# Quản Lý Phòng Khám - Frontend

Dự án frontend cho hệ thống Quản Lý Phòng Khám.

## Cài đặt và Chạy

1. Cài đặt các thư viện phụ thuộc:
    ```bash
    npm install
    # hoặc
    yarn install
    ```

2. Chạy ứng dụng trên môi trường phát triển:
    ```bash
    npm run dev
    # hoặc
    yarn dev
    ```

3. Build cho môi trường production:
    ```bash
    npm run build
    # hoặc
    yarn build
    ```
